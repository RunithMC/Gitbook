# Commands, Permissions & Placeholders

## Admin Commands (/rtrades)

{% hint style="success" %}
**Grant them all permissions for admin commands**

`runithtrades.admin.*` or `runithtrades.admin.use` for /rtrades open usage
{% endhint %}

| Command                    | Description                             | Permissions               |
| -------------------------- | --------------------------------------- | ------------------------- |
| `open (player) (category)` | Open a trade menu category for an user. | runithtrades.admin.open   |
| `reload`                   | Reload the plugin configuration files.  | runithcrates.admin.reload |
