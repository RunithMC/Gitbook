### Features

* Matrix Layout
* Unlimited Rewards&#x20;
* Robust Massive Opening System
* Mass Open
* Virtual Keys
* Preview Inventory
* Crates Locations
* PlaceholderAPI Hook

### Requeriments

* Java +21
* Software Version: +1.8.9
* Database MongoDB (Optional)
* [RunithCore](/broken/pages/PbYb0GukRhiS4qCHdRal)
